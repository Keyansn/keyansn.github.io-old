C++ sudoku solver

Multiple algorithm comparrison for solving sudoku puzzles

Overview

In first year, the final programming coursework was to implement a multitude of algorithms for solving sudoku puzzles. Each having their own benefits and drawbacks with some being practical to use and others being completely unfeasible in some situation. This project was written in C++ and ultimately it was concluded that backtracking was the optimal solution to this problem.

The report and code is available here

Skills
